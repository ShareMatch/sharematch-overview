## Issues Found & Fixed

### ISSUE-001: Mobile horizontal overflow [medium, fixed]
- **Category:** performance / visual
- **Source:** browser (375x812 mobile emulation)
- **Evidence:** `document.documentElement.scrollWidth` = 391px vs 375px viewport
- **Root cause:** Fixed `<nav>` element and Platform section's `absolute -inset-10` blur glow div extending 16px beyond viewport
- **Fix:** Added `overflow-x: hidden` to html/body in globals.css; added `overflow-hidden` to blur glow div
- **Verified:** Post-fix scrollWidth = 375px (matches viewport exactly)

### ISSUE-002: Hero "Create an Account" linked to #join anchor [low, fixed]
- **Category:** functional
- **Source:** browser / a11y tree
- **Evidence:** Hero CTA pointed to `#join` anchor while CTA section's identical button linked to `https://rwa.sharematch.me`
- **Fix:** Updated hero button to link to `https://rwa.sharematch.me` with `target="_blank"`
